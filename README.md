# 🚀 Object Detection and Tracking using YOLOv8

## 📌 Project Overview
This project is a real-time Object Detection and Tracking system built using YOLOv8 and OpenCV.  
It detects objects from webcam/video, tracks them with unique IDs, and performs basic analytics like person counting, FPS calculation, and alert system.

It is developed as part of the **CodeAlpha Internship Project**.

---

## 🎯 Features
- Real-time object detection using webcam
- YOLOv8 pretrained model (YOLOv8n)
- Object tracking with unique IDs
- Person detection and counting
- FPS (Frames Per Second) display
- Alert system for detection
- Snapshot saving feature
- Email notification system (optional)
- Video recording output

---

## 🛠️ Technologies Used
- Python
- OpenCV
- Ultralytics YOLOv8
- smtplib (Email alerts)
- NumPy
- Streamlit (optional UI)
